Get a Mapbox Access Token (Free)

Metainfo Mapper works perfectly with just a blank canvas, but you can optionally use this token to stream high‑quality hybrid satellite maps as a background for your JPG, TIFF, PNG, and RAW image data.

Sign up for Mapbox (free) Open your Access Tokens Back to Metainfo Mapper

1) Create a free Mapbox account (Optional)

If you'd like to add satellite imagery as a background, visit the Mapbox sign‑up page and create a free account. You can use a personal email and choose the free tier to start. This step is completely optional - the app works great without it!

Link: account.mapbox.com/auth/signup/

2) Open the Access Tokens page

After signing in, navigate to Account → Access tokens. Mapbox will automatically create a Default public token for you.

Link: account.mapbox.com/access-tokens/

3) Create (optional) a dedicated token

You can click Create a token to generate a dedicated token for Metainfo Mapper. Keep the default public scopes, and optionally set URL restrictions for your domain later.

4) Copy the token and paste into Metainfo Mapper

Click the copy icon next to your public token, return to the app, and paste it under Basemap Token → Use Mapbox Token. Your browser stores the token locally and never uploads it to the server. This will add Mapbox satellite imagery as a background to your existing blank canvas display.

5) Load images to activate hybrid satellite tiles

Drag photos into the app. Once GPS bounds are known, Metainfo Mapper fetches only tiles for the flight area (zoom 15–18). You can clear the token any time to return to the blank canvas. Remember: the app works perfectly without any token at all!

Tips & Limits

  • No token needed: Metainfo Mapper works great with just the blank canvas - all GPS plotting and KML export features work perfectly without Mapbox.
  • Use the default mapbox/satellite-streets-v12 style for hybrid satellite + labels.
  • Tokens are public; restrict by URL in Mapbox if publishing widely.
  • Free tier has request limits; Metainfo Mapper minimizes calls using buffered bounds.
