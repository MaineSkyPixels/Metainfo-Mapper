Get a Mapbox Access Token (Free)

Use this token in Metainfo Plotter to stream high‑quality hybrid satellite maps.

Sign up for Mapbox (free) Open your Access Tokens Back to Metainfo Plotter

1) Create a free Mapbox account

Visit the Mapbox sign‑up page and create a free account. You can use a personal email and choose the free tier to start.

Link: account.mapbox.com/auth/signup/

2) Open the Access Tokens page

After signing in, navigate to Account → Access tokens. Mapbox will automatically create a Default public token for you.

Link: account.mapbox.com/access-tokens/

3) Create (optional) a dedicated token

You can click Create a token to generate a dedicated token for Metainfo Plotter. Keep the default public scopes, and optionally set URL restrictions for your domain later.

4) Copy the token and paste into Metainfo Plotter

Click the copy icon next to your public token, return to the app, and paste it under Basemap Token → Use Mapbox Token. Your browser stores the token locally and never uploads it to the server.

5) Load images to activate hybrid satellite tiles

Drag photos into the app. Once GPS bounds are known, Metainfo Plotter fetches only tiles for the flight area (zoom 15–18). You can clear the token any time.

Tips & Limits

  • Use the default mapbox/satellite-streets-v12 style for hybrid satellite + labels.
  • Tokens are public; restrict by URL in Mapbox if publishing widely.
  • Free tier has request limits; Metainfo Plotter minimizes calls using buffered bounds.
